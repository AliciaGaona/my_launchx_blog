2. REPL (Read-Evaluate-Print-Loop).

¿Qué es? Es similar a Shell (Unix / Linux) y al símbolo del sistema. si podemos interactaar a través de salidas de comandos/expresiones. Es útil para escribir y depurar los códigos. El trabajo de REPL se puede entender en su forma completa:

Leer : lee las entradas de los usuarios y las analiza en la estructura de datos de JavaScript. Luego se almacena en la memoria. Eval : La estructura de datos de JavaScript analizada se evalúa para los resultados. Imprimir : el resultado se imprime después de la evaluación. Bucle : repite el comando de entrada. Para salir de NODE REPL, presione ctrl + c dos veces
